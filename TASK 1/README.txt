 Task 1 – To-Do List (Command Line Application)

This is a Python-based command-line To-Do List application developed as part of the CodSoft Python Programming Internship – June 2025.

The application allows users to manage tasks by adding, viewing, marking as completed, and deleting them. It also includes data persistence using JSON and additional features for an interactive experience.

 Features

- Add new tasks
- View all tasks with completion status
- Mark tasks as done
- Delete tasks
- Save and load tasks using a JSON file
- Track user XP (experience points) for completing tasks
- Terminal-based user interface with color highlights and progress indication
- Easter egg command for bonus XP and dynamic messages
- Mood-based startup interaction

Technologies Used

- Python 3.11+
- Built-in modules: `json`, `os`, `random`, `time`
- External libraries:
  - `colorama` for terminal color output
  - `tqdm` for progress/loading effects
