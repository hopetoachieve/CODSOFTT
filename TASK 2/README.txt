Task 2 – GUI Calculator (Tkinter)

This is a GUI-based calculator application built using Python and Tkinter as part of the CodSoft Python Programming Internship – June 2025.

The calculator supports basic arithmetic operations, clear input, and real-time keyboard input (including number pad). It provides a clean and responsive user interface and handles errors such as division by zero or invalid expressions.

Features

- Graphical User Interface using Tkinter
- Supports addition, subtraction, multiplication, and division
- Interactive input using both on-screen buttons and keyboard keys
- Input validation and error handling
- Backspace (`←`) and clear (`Esc`) key support

 Keyboard Support

| Key         | Function             |
|-------------|----------------------|
| `0-9`       | Add number to input  |
| `+ - * / .` | Operators and decimal|
| `Enter`     | Evaluate expression  |
| `Backspace` | Remove last character|
| `Escape`    | Clear input field    |

 Technologies Used

- Python 3.11+
- Tkinter (standard GUI library, no installation needed)

How to Run

1. Make sure Python 3 is installed.
2. Open terminal in the `Task2-Calculator/` folder.
3. Run the script:
   ```bash
   python calculator_gui.py
